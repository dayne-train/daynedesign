# dayne.design

Personal portfolio site for Dayne Petera — product designer with 10+ years of experience.

**Live site:** [dayne.design](https://dayne.design)

## Stack

Pure HTML, CSS, and JavaScript — no frameworks, no build tools. Hosted on GitHub Pages with a custom domain.

## Structure

- `index.html` — main portfolio page (case study grid, experience, fish quiz, testimonials)
- `*-case-study.html` — 14 individual case study pages
- `global.css` — shared stylesheet
- `assets/` — favicon, resume PDF, company logos
