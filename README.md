# dayne.design

Personal portfolio site for Dayne Petera — product designer with 10+ years of experience.

**Live site:** [dayne.design](https://dayne.design)

## Stack

Pure HTML, CSS, and JavaScript — no frameworks, no build tools. Hosted on GitHub Pages with a custom domain.

## Structure

- `index.html` — main portfolio page (case study grid, experience, fish quiz)
- `retro.html` — 2001-era easter egg page (accessible via footer year menu)
- 14 case study pages (`hris-sync.html`, `lessen-pro.html`, `veho.html`, etc.)
- `global.css` — shared stylesheet
- `assets/` — favicon, resume PDF, company logos, images
